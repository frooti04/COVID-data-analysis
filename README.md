# COVID-data-analysis
COVID analysis using Excel
## Objective
To analyze COVID-19 data and identify trends in cases, recovery and deaths.
## Dataset
Date, Country, Confirmed Cases, Deaths, Recovered, Active Cases, etc.
## Tools Used 
Microsoft Excel
## Analysis Performed
-Actice cases calculation
-Recovery rate and death rate analysis
-Country wise comparison
-Trend analysis using Pivot Tables
